# web-dev_parcial1

## Getting started:

**For the back-end:**

```
cd parcial1
yarn install
yarn start
```

**For the front end:**

```
cd ./parcial1/front
yarn install
yarn start
```

For mongo: 

Make sure mongodb is up and running. For linux:
```
service mongod start
```
